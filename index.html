ใช้คำนวณเปอร์เซ็นต์บรรลุเป้าและยอดที่ยังขาดบนหน้า Dashboard
เป้ายอดขายรายเดือน

กด "เฉลี่ยจากเป้าปี" เพื่อแบ่งเป้าปีเท่าๆ กัน 12 เดือน · แก้รายเดือนเองได้ เว้นว่าง = ยังไม่กำหนด

วาง URL ที่ได้จากการ Deploy โค้ด Apps Script (ดูขั้นตอนด้านล่าง)
วิธีติดตั้ง (ทำครั้งเดียว):
  1. สร้าง Google Sheet ใหม่ → เมนู ส่วนขยาย › Apps Script
  2. วางโค้ดจากไฟล์ AppsScript.gs ที่ให้มา แล้วบันทึก
  3. กด Deploy › New deployment › Web app
  4. ตั้ง Execute as: Me, Who has access: Anyone
  5. คัดลอก URL ที่ได้ มาวางในช่องด้านบน
ข้อมูลจะถูกบันทึกลงเครื่องเสมอ (ใช้งานได้แม้ไม่ต่อเน็ต) และซิงก์ขึ้น Sheet อัตโนมัติเมื่อเชื่อมต่อแล้ว
สำรองข้อมูลในเครื่อง

ไฟล์ JSON เก็บโครงการ รายการสินค้า สถานะ และประวัติกิจกรรมครบถ้วน โดยไม่เก็บ URL เชื่อมต่อ Google Sheet
